# Mus — Juego de cartas

Juego de **Mus** para 4 jugadores: **tú** (Sur) con compañero **Norte** contra **Este** y **Oeste** (bots).

Las cartas se muestran **visibles** en la mesa para todos los jugadores.

## Cómo jugar

1. Abre **`index.html`** con doble clic (Chrome, Edge o Firefox).
2. Pulsa **Nueva partida**.
3. En cada mano:
   - **Mus / No mus**: acordáis si jugar o repartir de nuevo.
   - **Cambio**: descarta cartas que no te interesen (clic para seleccionar).
   - **Apuestas**: Grande, Chica, Pares y Juego — Envido, Doble, Hacer, Paso u Órdago.

Primer equipo en **40 piedras** gana.

## Baraja

Baraja española de 40 cartas: Oros, Copas, Espadas y Bastos (As, 2–7, Sota, Caballo, Rey).

## Ejecutar con servidor local (opcional)

```bash
npx --yes serve .
```

Luego abre la URL que indique (normalmente http://localhost:3000).
